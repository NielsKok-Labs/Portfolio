Pryvee

2025 – heden In actieve ontwikkeling SaaS M365 Cloudflare Workers Stripe Graph API

Pryvee is een M365 governance en privacy SaaS platform dat ik zelf bouw — volledig als static site op GitHub Pages met Cloudflare Workers als backend. Gericht op IT-teams, MSPs en compliance-verantwoordelijken die hun Microsoft 365-omgeving willen auditen en AVG-compliant willen werken, zonder zware infrastructuur of agents te installeren.

01

Waarom Pryvee

Veel MKB-organisaties en MSPs hebben geen inzicht in de beveiligingsstatus van hun Microsoft 365-tenant — maar de bestaande tooling is te zwaar, te duur of vereist Azure-infrastructuur. Pryvee vult die gap: lichtgewicht, geen installatie, geen footprint in de tenant van de klant.

Het idee ontstond vanuit mijn eigen praktijk als M365 consultant. Ik voerde regelmatig handmatige health checks uit en zag dat er geen toegankelijk, betaalbaar alternatief was voor MKB en MSPs. Pryvee is het antwoord daarop — gebouwd met de technologie die ik dagelijks gebruik.

Geen infrastructuur in de tenant van de klant

Pryvee draait volledig als SaaS. Geen agents, geen scripts, geen Azure-resources in de klantomgeving — alleen leestoegang via de Microsoft Graph API met MSAL.

02

Architectuur

De architectuur is bewust zo eenvoudig en kostenefficiënt mogelijk gehouden. De frontend is een pure static site op GitHub Pages — geen framework, geen buildstap. De backend bestaat uit Cloudflare Workers die de licentievalidatie, Stripe-webhooks en Graph API-proxying afhandelen. Licentiedata wordt opgeslagen in Cloudflare KV.

FE GitHub Pages

Pure HTML/CSS/JS static site. Geen framework, geen buildstap, geen kosten. MSAL.js voor Microsoft authenticatie.

BE Cloudflare Workers

Edge-runtime voor licentievalidatie, Stripe webhook verwerking en Graph API proxy. KV voor licentieopslag per product en e-mail.

💳 Stripe

Payment links per plan met metadata. Webhook naar Worker bij geslaagde betaling. Licentie wordt per product opgeslagen: snapshot:email, redact:email.

03

Modules

Pryvee is opgebouwd uit losse modules die elk een specifiek governance- of privacy-probleem oplossen. Klanten kunnen starten met één module en uitbreiden naar de volledige suite.

📊
Snapshot — CIS M365 audit Live

Verbind je tenant via MSAL, scan op basis van de CIS M365 Benchmark en ontvang binnen minuten een managementklaar PDF-rapport. Geautomatiseerde checks op MFA, Conditional Access, gastaccounts, Secure Score en extern delen. Geen installatie, alleen-lezen. €249 eenmalig of €79/mnd.

🔒
Redact — In-browser PII redactie Live

Persoonsgegevens verwijderen uit PDF en DOCX — volledig in de browser, niets verstuurd naar servers. NER-engine herkent BSN, IBAN, namen, adressen, telefoonnummers en meer op basis van Nederlandse context. AVG-compliant by design. €39/mnd per organisatie.

🏗️
Spaces — Workspace provisioning Binnenkort

Browser-based M365 workspace provisioning via MSAL en Graph API — een lichtgewicht alternatief voor Power Platform of Provision Assist, zonder infrastructuur in de klantomgeving. €39/mnd.

04

Technische keuzes

De Graph API-calls worden gerouted via een Cloudflare Worker proxy (/graph-proxy) met een whitelist van toegestane endpoints. CORS is gelocked op specifieke domeinen. Stripe webhook-validatie verloopt via HMAC-signing. Rate limiting is ingericht via een aparte KV-namespace.

Licenties worden product-gescheiden opgeslagen in KV: snapshot:email en redact:email. Het Essentials-pakket schrijft beide keys tegelijk bij activatie. Een kortlevend HMAC-signed session token zorgt dat licentievalidatie niet afluisterbaar is.

De NER-engine in Redact is regex-gebaseerd met Nederlandse context — BSN met elfproefvalidatie, IBAN in NL-format, adressen alleen met bijbehorende postcode, namen uitsluitend bij expliciete triggers. De stap naar een WASM-model (spaCy of flair via Pyodide) staat op de roadmap voor hogere nauwkeurigheid.

05

Go-to-market

De primaire strategie is MSP-first: managed service providers kunnen Pryvee aanbieden aan hun klanten met een reseller-marge van 30–40%. Voor MSPs is er een white-label optie met eigen logo op de PDF-rapporten en een multi-tenant dashboard.

Voor MKB-klanten zijn de voornaamste triggers: cyberverzekering-vereisten, NIS2 supply chain-druk en ISO 27001/BIO-certificeringsprocessen. Het instapmodel is een 14-daagse gratis trial of een gratis eerste scan — geen prijskorting maar drempelverlagende toegang.

  • Essentials bundle (Snapshot + Redact + Spaces) €149/mnd
