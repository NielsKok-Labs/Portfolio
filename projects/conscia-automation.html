Stage · Security & Network Automation

Network Automation
Toolbox | Conscia

Februari – Juni 2025 Succesvol afgerond Python REST API Check Point F5 Automation Security

Tijdens mijn stage bij Conscia Nederland ontwikkelde ik een modulaire Python-toolbox voor het automatiseren van netwerkconfiguraties binnen de Security- en NOC-omgeving. Via REST API-integraties met Check Point firewalls en F5 load balancers werd repeterend handmatig werk geautomatiseerd — met significante tijdsbesparing in testscenario's als resultaat.

01

Context & opdracht

Conscia is een Europese IT-dienstverlener gespecialiseerd in netwerken, security en cloud. Binnen de Security- en NOC-afdeling werden veel netwerkconfiguraties handmatig uitgevoerd — een tijdrovend en foutgevoelig proces bij complexe infrastructuur met meerdere vendors.

De opdracht: een herbruikbare Python-toolbox bouwen die veelvoorkomende configuratietaken automatiseert via de REST API's van de aanwezige netwerkplatformen. De toolbox moest modulair zijn zodat nieuwe vendor-integraties eenvoudig toe te voegen zijn.

02

De toolbox — modulaire opzet

De toolbox is opgezet als een modulaire Python-bibliotheek waarbij elke vendor een eigen module heeft. Zo blijven vendor-specifieke implementatiedetails geïsoleerd en is de toolbox eenvoudig uitbreidbaar naar nieuwe platforms.

Check Point

REST API-integratie met Check Point Management API. Automatisering van firewall-regelconfiguraties, objectbeheer en beleidswijzigingen.

F5 Load Balancers

REST API-integratie met F5 BIG-IP iControl REST. Automatisering van load balancer-configuraties, virtual servers en pool-beheer.

Python Core

Modulaire opzet met gedeelde utilities voor authenticatie, sessie-management, foutafhandeling en logging — herbruikbaar over alle vendor-modules.

03

Technische aanpak

1
API-inventarisatie & authenticatie

Analyse van de beschikbare REST API-endpoints voor Check Point en F5. Sessie-authenticatie geïmplementeerd met token-management en automatische vernieuwing — zodat langlopende scripts niet halverwege uitloggen.

2
Modulaire bibliotheekstructuur

Elke vendor krijgt een eigen Python-module met een consistente interface. Gedeelde utilities voor HTTP-requests, foutafhandeling en respons-parsing zitten in een core-laag — vendor-modules bouwen hierop voort.

3
Automatisering van configuratietaken

Veelvoorkomende handmatige taken omgezet naar herbruikbare functies: aanmaken en wijzigen van firewall-objecten, beleidswijzigingen doorvoeren, load balancer-pools aanpassen en configuraties valideren.

4
Testscenario's & validatie

De toolbox is ingezet in testscenario's binnen de NOC-omgeving. Configuraties die eerder handmatig tientallen minuten kostten, werden geautomatiseerd uitgevoerd — met significante tijdsbesparing als aantoonbaar resultaat.

04

Resultaat

Minder handmatige fouten

Configuratietaken die eerder handmatig uitgevoerd werden zijn nu geautomatiseerd — met consistente uitvoering en ingebouwde validatie.

Tijdsbesparing

Significante tijdsbesparing in testscenario's. Repeterend werk dat engineers handmatig uitvoerden wordt nu in seconden afgehandeld.

Herbruikbaar & uitbreidbaar

Modulaire opzet maakt toevoegen van nieuwe vendor-integraties eenvoudig zonder de bestaande code te raken.

Van security-consultant naar automatiseerder

Dit project liet zien dat de combinatie van security-kennis en Python-automatisering directe waarde heeft in een NOC-omgeving. Wie begrijpt wat een firewall-policy doet, bouwt betere automatisering dan wie alleen de API kent.
